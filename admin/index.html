Code Anywhere Instructions

When editing the static pages on Code Anywhere, right click the container name (aktsa) and select "SSH Terminal". Type git pull and press enter before beginning development to download the latest changes.

When finished making edits, go back to the terminal and type git add -A and press enter, then type git commit -m "INSERT SHORT DESCRIPTION OF CHANGES HERE" and press enter, followed by git push.
